# Pagina Web de Servicios

Proyecto de pagina web orientada a la presentacion de servicios profesionales.  
Esta aplicacion esta desarrollada con **Vite** y **TypeScript**, utilizando HTML y CSS para la maquetacion y los estilos.

## Caracteristicas

- Estructura moderna basada en Vite.
- Maquetacion en varias secciones para mostrar servicios, informacion y contacto.
- Diseno responsive compatible con dispositivos moviles y escritorio.
- Codigo organizado dentro de la carpeta `src`.
- Version optimizada para produccion generada en la carpeta `build`.

## Tecnologias utilizadas

- **TypeScript**
- **HTML5**
- **CSS3**
- **Vite**
- **Node.js** + **npm**

## Requisitos previos

Asegurate de tener instalado:

- Node.js (version LTS recomendada)
- npm

Comandos para confirmar versiones:

node -v
npm -v

## Instalacion

Clona el repositorio:

git clone https://github.com/J4nk37/Pagina-Web-Servicios.git


Accede al proyecto:

cd Pagina-Web-Servicios


Instala las dependencias:

npm install

Scripts disponibles
npm run dev

Inicia el servidor de desarrollo:

npm run dev


Abre en el navegador la URL indicada (normalmente http://localhost:5173/).

npm run build

Genera la version para produccion:

npm run build


Los archivos finales se guardan en build/.

npm run preview

Sirve localmente la build generada:

npm run preview

## Estructura del proyecto

Pagina-Web-Servicios/
├─ build/             # Archivos finales de produccion
├─ node_modules/      # Dependencias instaladas
├─ src/               # Codigo fuente (TS, estilos, etc.)
├─ index.html         # Punto de entrada
├─ package.json       # Configuracion y scripts del proyecto
├─ package-lock.json  # Bloqueo de dependencias
└─ vite.config.ts     # Configuracion de Vite

## Despliegue

Tras ejecutar:

npm run build


Los archivos de la carpeta build/ pueden subirse directamente a cualquier hosting estatico:

-GitHub Pages

-Netlify

-Vercel

-Servidor propio
